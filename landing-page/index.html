Powerful Features

Everything you need for mobile app security testing and reverse engineering

SSL Pinning Bypass

Bypass SSL certificate pinning with built-in scripts for OkHttp3, Conscrypt, WebView, Flutter, and React Native

APK Patcher

Inject bypass scripts directly into APKs without requiring rooted devices. Works with Frida Gadget

Real-time Capture

Live HTTP/HTTPS traffic interception with detailed request/response analysis and timing information

Request Mapping

Map URLs to different destinations with regex pattern support. Perfect for API testing and debugging

Mock Responses

Create custom mock responses with status codes, headers, and bodies. Test error scenarios easily

Frida Integration

Dynamic code injection and runtime hooking with Frida server integration for advanced testing

Advanced Filtering

Filter by method, status code, host, URL, size, duration, and content type with regex support

Export Options

Throttle Control

Simulate network conditions with bandwidth limiting and latency injection for performance testing
